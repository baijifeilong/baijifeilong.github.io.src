# Bash大杂烩

## 用函数切换代理

```bash
function freedom {
    export HTTP_PROXY=localhost:8118
    export HTTPS_PROXY=localhost:8118
}

function slavery {
    unset HTTP_PROXY
    unset HTTPS_PROXY
}
```

将以上两个函数添加到`~/.zshrc`，然后`source ~/.zshrc`即可使用命令`freedom`和`slavery`切换代理
